# COVID-19 Global Dashboard (Tableau)

## Project Overview

Developed an interactive Tableau dashboard to analyze global COVID-19 cases, deaths, and trends across countries and regions. The dashboard provides insights into the spread and impact of the pandemic using data visualization techniques.

## Tools Used

* Tableau
* Excel
* Data Visualization

## Objectives

* Track COVID-19 confirmed cases and deaths.
* Compare pandemic trends across countries.
* Identify regions with the highest impact.
* Present data through interactive visualizations.

## Key Insights

* Visualized country-wise COVID-19 cases and deaths.
* Analyzed trends over time.
* Compared the impact of COVID-19 across different regions.
* Highlighted countries with the highest infection rates.

* Data Analysis

* Dashboard Development
* Tableau Reporting
* KPI Monitoring
