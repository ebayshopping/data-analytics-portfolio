# Sales Analytics Dashboard

## Overview
Developed an interactive Power BI dashboard to analyze sales performance, profitability, product performance, and order trends.

## Tools Used
- Power BI
- Power Query
- DAX

## KPIs
- Total Sales: 9.57M
- Total Profit: 8.84M
- Order Quantity: 4,384

## Insights
- Identified top-performing products.
- Analyzed sales trends over time.
- Compared budget vs actual sales performance.
- Evaluated profitability across products.
