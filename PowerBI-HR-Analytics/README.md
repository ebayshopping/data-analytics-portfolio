# HR Analytics Dashboard

## Overview
Created an HR dashboard to analyze employee attrition and workforce performance.

## Tools Used
- Power BI
- DAX
- Power Query

## KPIs Tracked
- Attrition Rate
- Employee Count
- Job Satisfaction
- Average Salary

## Key Insights
- Higher attrition observed among employees with lower job satisfaction.
- Certain departments had significantly higher turnover.
