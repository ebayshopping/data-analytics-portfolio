## Overview
Analyzed retail sales data to identify seasonal trends and marketing opportunities.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Analysis Performed
- Data Cleaning
- Trend Analysis
- Sales Forecasting
- Customer Analysis

## Key Insights

- Marketing campaigns increased revenue significantly.
