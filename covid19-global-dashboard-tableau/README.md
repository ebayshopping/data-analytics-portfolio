# COVID-19 Global Dashboard (Tableau)

## Overview

This project presents an interactive Tableau dashboard that visualizes global COVID-19 cases and deaths across different countries and regions. The dashboard helps identify trends and provides insights into the impact of the pandemic.

## Tools Used

* Tableau
* Microsoft Excel
* Data Visualization

## Objectives

* Analyze COVID-19 confirmed cases and deaths.
* Compare country-wise pandemic impact.
* Visualize trends using interactive dashboards.
* Generate insights from global COVID-19 data.

## Key Insights

* Identified countries with the highest number of cases and deaths.
* Compared COVID-19 trends across regions.
* Analyzed changes in cases over time.
* Visualized pandemic impact using interactive charts and maps.
